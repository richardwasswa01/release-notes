March 30, 2026
v 1.0.0
Incremental Improvements to Core Modules
1. Login Screen:
  • Cookies are integrated to the platform.
2. Vitals Screen:
2.1. Notification Sidebar:
  • When a notification is clicked, users now are directed to the resident profile.
    • When a notification is marked as reviewed, it can be checked later in resident profile
3. Residents Screen:
3.1. Individual Thresholds:
  • All the edit changes can be saved without an issue now. E.g. change individual thresholds from global settings to manual and make the duration of out of bed 100 minutes.
  • When the Global setting is selected in Individual Threshold settings, any change made to the Global settings is automatically applied to the Individual Threshold screen.
3.2. Resident Overview:
  • Adding a new location to a resident without a device now shows a warning pop up.
3.3. Trends:
  • Vertical max. values are now calculated as +5 on the threshold lines.
  • Night report values formatting changed for English and German. E.g. English: 1h 2m 3s, Deutsch: 1 h 2 min 3 s
4. Notifications Screen:
Bed Exit Notifications:
  • Time period editing functionality changed with new dropdown component implementation.
  • Threshold activity is now seen in view mode with an Active badge.
  • Filtering the notifications based on Type, Resident ID, Device ID, Date Range and Status is now available.
